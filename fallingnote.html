Falling Notes

Max for Live device that allows for real time performance of musical score via falling MIDI notes

Beta Release

September 16, 2017 by Ian

In conjunction with Albany Medical Center, I have been working on accessibility software for people with limited upper body movement. Through this rewarding job, I have been able to help make the disabled community a better place full of creative music and fun to use devices.

This piece of software is just one of my many creations for the company. The current version of FallingNote is the intellectual property of My Music Machines Inc., and unfortunately it is not available for sale at the moment. However, I can give a high level explanation on how it works.

Fallingnote is a Max for Live Device that uses the Ableton API to read in MIDI data from the track that the user places it on. It then automatically translates that data using Processing for JavaScript to display notes onscreen that are auto quantized to be in the correct time signature and tempo to be played along with a backing track or band as they fall down the screen to the corresponding keys on a visual representation of a keyboard. Any VST can be placed on the same MIDI track so the user can be able to hear the notes that they are playing.

While working on this project I worked alongside Medical professionals at Albany Medical Center and highly experienced and knowledgeable RPI students to create other accessibility software for people with spinal cord injuries who are interested in playing, creating, or learning music. Kathryn Lovell worked on the machine code for the Jamboxx so that way it would interact with my program and Ableton, and Robert Smock contributed to the CAD design of the controller.
