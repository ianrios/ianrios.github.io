Grandulator

February 6, 2017 by Ian

Version 1 now available!(UI improvements on the way)

This program is for ensemble performances that allows for real time autotuning of live granular synthesis.

Borilliant

May 18, 2017 by Ian

Beta version now available!

This program is for class scheduling based on professor rankings using the YACS Database provided by RCOS at RPI and student ratings at the end of each semester.

Barely Enough Ingredients

August 29, 2017 by Ian

In development, no version avaliable at this time

Website dedicated to storing a client’s ingredients in a database, and searching for suitable recipes that fit the client’s needs. In addition, it searches nearby grocery stores that are within the client’s budget in the event they need more ingredients for a particular recipe.

Uses Ruby on Rails.

MIDI Networking

October 16, 2017 by Ian

Version 1 now available for Transport and Note!

Falling Note

June 15, 2017 by Ian

Completed, but currently not for public use
