Score Reader

Max for Live device that allows for real time performance of musical score via MIDI notes

Version 1 for download now!

October 16, 2017 by Ian

In conjunction with Rensselaer Polytechnic Institute, I, along with Nathan K, Kathryn Lovell, and Robert Smock have been researching new methods of musical collaboration, specifically creating accessibility software for people with restrictive movement and mobility. Through this research position, we came up with two methods for sending midi information over a network via UDP.

These pieces of software barely scratch the surface of the posibilities that can be done with network communications with music. The current versions of the MIDI and Transport Send and Recieve patches only work for Ableton Live using Max for Live. The MIDI send and recieve patches can be used to send note and velocity data from one machine to another, which allows for a master computer to only output sound while a client computer does all the work of performing without the latency of running many vsts and effects.

The Transport send and recieve patches are used for sending time signature, tempo, and start time to many computers, so that everyone playing in an ensemble can be in time. This device syncs to the Ableton API to pack and send accurate data from the track that the user places it on.

These patches are meant to be used with the Score Reader.

Future plans are to create a world server to collaborate over multiple networks, and combine the patches for ease of use.
